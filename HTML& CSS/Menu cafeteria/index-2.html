EXPLICACIÓN DEL HTML


HEADER



Esta etiqueta la utilice durante el diseño y desarrollo de la página web "Menú Cafeteria" para elaborar el encabezado del menú y mostrar la información relacionada con la empresa o marca: como su nombre y ubicación por ejemplo. Esta etiqueta la puse dentro de una etiqueta main que más adelante explicaré para qué la utilice.


FOOTER



La etiqueta Footer fue utilizada durante el diseño y desarrollo de la página "Cafeteria ADSO" para elaborar el pie de página del menú donde el usuario podrá encontrar información de contacto de la empresa y número de la version de la página web para mayor conocimiento del cliente.


MAIN



La etiqueta Main la utilice en la pagina de "Cafeteria ADSO" para crear el cuadro principal que contendrá el encabezado,los titulos, parrafos, iconos, imagenes, enlaces y también el pie de página.

La etiqueta Main sirve para representar el contenido principal del cuerpo o aquel que está directamente relacionado con el tema central de un documento o aplicación, que en este caso es un menú para la cafeteria ya mencionada.


SECTION



Usé la etiqueta "section" en la página para agrupar el contenido que está temáticamente relacionado. Por ejemplo, en mi página tengo dos secciones identificadas como "Café" y "Postres".

Cada una de estas secciones contiene un título, una imagen y un elemento "article" que contiene información más detallada sobre elementos específicos de ese tema. Al utilizar estas etiquetas, estoy ayudando a los motores de búsqueda y a los lectores de pantalla a comprender mejor la estructura y el contenido de mi página.


ARTICLE



La etiqueta "article" la utilicé en la página de "Cafeteria ADSO" para definir contenido independiente y autocontenido. En mi caso, dentro de cada sección ("Café" y "Postres"), tengo elementos "article"que contienen listas de productos, como "cafés" y "postres", junto con sus precios correspondientes.

Cada uno de estos elementos "article" es independiente y tiene sentido por sí mismo.
